A simple custom block plugin to embed and display a styled resume section on any WordPress page or post.

== Description ==

The **Nikhil Plugin** adds a reusable Gutenberg block that lets you display a resume directly on any WordPress page or post. It allows customization of your job title, education, experience, and contact details in a clean layout using the WordPress block editor.

This plugin was scaffolded using the `@wordpress/create-block` tool.

- Clean layout
- Gutenberg block compatible

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/nikhil-plugin` directory, or install the plugin via WordPress admin dashboard.
2. Activate the plugin through the **Plugins** screen in WordPress.
3. Use the block editor to insert the **Nikhil Resume Block**.
4. Fill in your resume details and publish!

== Frequently Asked Questions ==

= How do I use the plugin? =

After activating, go to any page/post editor and insert the block titled **"Nikhil Resume Block"**. Customize your resume content right inside the editor.

= Can I use this with any theme? =

Yes! It works with all modern block-based themes.
